What it does

FancyZones still snaps your windows into zones. Fancy Schmancy Zones remembers each arrangement and brings it back — so a whole layout becomes one thing you can toggle to.

🔒

Lock a layout

Capture every window exactly where it sits and save it under a name like “Coding” or “Email.”

🔁

Flip between layouts

One hotkey restores a layout — each window jumps back to its spot and comes to the front.

🎛️

Manage from the tray

Left- or right-click the tray icon to pick, rename, update, or delete any layout.

How it works

Three steps. No setup, no config files.

Arrange your windows

Use FancyZones (Shift-drag into zones) or just place windows where you like them.

Lock it

Press Ctrl+Alt+Shift+L and give the arrangement a name. Build as many as you want.

Flip anytime

Double-tap the Ctrl key to flip to the next layout — windows snap back into place. (Prefer a hotkey? Ctrl+Alt+Shift+Q works too.)

Everything from the system tray

No window to keep open — it lives by your clock. Left- or right-click the icon for the full menu: your saved layouts, plus lock, manage, and flip.

Your layouts, one click away

Every layout you’ve locked shows up by name — click to jump straight to it.

This is the actual menu the app shows. The little grid icon sits near your clock (you may need to click the ^ arrow to see it).

Hotkeys

They use a low-level keyboard listener, so they keep working even when other utilities have grabbed common shortcuts. Prefer the mouse? Everything’s in the tray menu too.

Flip to next layout  easiestDouble-tap Ctrl
Flip to next layoutCtrl + Alt + Shift + Q
Flip to previous layoutCtrl + Alt + Shift + W
Lock current layoutCtrl + Alt + Shift + L

Your layouts are stored in %APPDATA%\FancySchmancyZones\layouts.json.

Install

A normal one-click installer. Installs just for you (no admin), and the app is self-contained — you don’t need to install anything else.

Download

Grab FancySchmancyZones-Setup from the latest release.

Run it

It creates a Start Menu shortcut. You can tick “Start automatically when I sign in.”

Look for the tray icon

The little grid icon near your clock (under the ^ arrow). Click it to begin.

⚠ One current limitation — Windows Virtual Desktops

A layout only captures windows on the desktop you’re currently on. If you spread windows across several Windows virtual desktops, the app won’t see the ones on other desktops. Full multi-desktop support needs undocumented Windows features that change with every update — it’s on the roadmap, but it’s a hard problem. For now, one set of layouts per desktop works great.
