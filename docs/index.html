Small app, thoughtful details

It sweats the parts that make switching layouts feel right.

Lock any arrangement

Capture every window exactly where it sits and save it under a name like "Coding" or "Email." Build as many as you want.

Right Chrome profile, right spot

Layouts remember which Chrome or Edge profile each window used, and put each one back where it belongs.

A clean switch, every time

Flipping tucks away anything that isn't part of the layout, so the one you switch to is never left half-hidden.

Open a full workspace

Back after a reboot? One click opens the layout's apps that aren't running yet, then arranges them.

Never the wrong window

If a window a layout remembered is closed, its spot is simply left empty — it won't drag a random look-alike on top.

Lives in the tray

No window to keep open. It sits by your clock; everything's one click away in the menu.

Hotkeys

They use a low-level keyboard listener, so they keep working even when other utilities have grabbed common shortcuts. Prefer the mouse? It's all in the tray menu too.

One current limit — Windows Virtual Desktops

A layout captures windows on the desktop you're on. Spreading windows across several Windows virtual desktops isn't supported yet — it needs undocumented Windows internals. One set of layouts per desktop works great in the meantime.
